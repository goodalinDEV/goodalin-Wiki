# Команды

Тут вы сможете узнать о командах, которые есть на нашем сервере.

## Общие команды

| Команда                        | Описание                                                                                         |
| ------------------------------ | ------------------------------------------------------------------------------------------------ |                                             
| /sit                           | сесть на блок                                                                                    |
| /lay                           | лечь на блок                                                                                     |
| /skin set `nick`               | установить скин по нику                                                                          | 
| /sit playertoggle              | Включить/выключить режим сидения на вашей голове                                                 |
| /skin set `URL` `SLIM/CLASSIC` | установить скин по URL                                                                           |

## Чат

### РП-команды

| Команда                    | Описание                                                     |
| -------------------------- | ------------------------------------------------------------ |
| /&#60;g&#62;gme `действие` | Описать действие вашего персонажа                            |
| /&#60;g&#62;try `действие` | Испитать удачу                                               |
| /&#60;g&#62;do `описание`  | Описать ситуацию, которая происходит вокруг вашего персонажа |

g = написать в глобальный чат

### Личные сообщения

| Команда              | Описание                               |
| -------------------- | -------------------------------------- |
| /msg `ник сообщение` | Отправить личное сообщение игроку      |
| /r `сообщение`       | Ответить на последнее личное сообщение |
