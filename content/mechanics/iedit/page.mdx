# Кастомизация предметов

ItemEdit плагин - который позволяет вам менять название и описание предмета,
## Оформление

Для оформления предметов используеться [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/)

## Команды

### Название

| Команда                    | Описание                   |
| -------------------------- | -------------------------- |
| /ie rename `название` | Поменять название предмета |

### Описание

| Команда                              | Описание                        |
| ------------------------------------ | ------------------------------- |
| /ie lore set `строка` `текст` | Поменять строку в описании       |
| /ie lore del `строка `      | Удалить строку в описании |
