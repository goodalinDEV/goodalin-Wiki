# Быстрый бетон
На нашем сервере есть механика быстро превращать сухой бетон в обычный.

## Как это работает

Поставьте котёл
![](https://raw.githubusercontent.com/goodalinDEV/goodalin-Wiki/refs/heads/main/assets/images/fastconcrete/1.png)
Нажмите `ПКМ` по котлу, держа в руках сухой бетон, он превратится в обычный бетон.
![](https://raw.githubusercontent.com/goodalinDEV/goodalin-Wiki/refs/heads/main/assets/images/fastconcrete/2.png)
