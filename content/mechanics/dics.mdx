# Диски

(Доступно только для игроков с `GoodalinPlus`)

# Особенности дисков

- Требуется обычный диск для создания нового диска с музыкой
- Можно записать на козий рог
- Доступно только с модом PlasmoVoice

# Команды

| Команда                                 | Описание                                           |
| --------------------------------------- | -------------------------------------------------- |
| /disc burn `[URL на трек]` `<название>` | Создать музыкальный диск, надо держать в руке диск |
| /disc erase                             | Вернуть диск в обичное состояние                   |
| /disc search `[название]`               | Найти музыку по названию через YouTube             |
