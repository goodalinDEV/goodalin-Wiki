# Чат

## Основное

| Чат                   | Описание                                                     |
| -------------------------- | ------------------------------------------------------------ |
|  Локальный чат | 100 блоков вокруг игрока                         |
| Глобальный чат | Напишите `!` перед сообщением                                        |
### Команды:

| Команда                    | Описание                                                     |
| -------------------------- | ------------------------------------------------------------ |
| /&#60;g&#62;gme `действие` | Описать действие вашего персонажа                            |
| /&#60;g&#62;try `действие` | Испытать удачу                                               |
| /&#60;g&#62;do `описание`  | Описать ситуацию, которая происходит вокруг вашего персонажа |

## Компоненты чата:
| Компонент | Описание              |
| --------- | --------------------- |
| `@никнейм` | Упоминание игрока      |
| `[inv]`     | Показать инвентарь    |
| `[ender] `  | Показать эндер сундук |
| `[pos]`  | Показать координаты   |
| `[ping]`    | Показать свой пинг    |



### Личные сообщения

| Команда              | Описание                               |
| -------------------- | -------------------------------------- |
| /msg `ник сообщение` | Отправить личное сообщение игроку      |
| /r `сообщение`      | Ответить на последнее личное сообщение |
